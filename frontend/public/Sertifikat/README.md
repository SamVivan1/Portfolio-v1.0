# Sertifikat Portfolio

Folder ini berisi file sertifikat yang dapat diakses langsung dari website portfolio.

## Struktur File:
- `understanding-internet.pdf` - Understanding the Internet Certificate
- `apie-camp.pdf` - APIE Camp Certificate  
- `service-design.pdf` - Service Design Certificate
- `apie-advanced.pdf` - APIE Advanced Camp Certificate
- `junior-network-admin.pdf` - Junior Network Administrator Certificate

## Cara Menggunakan:
1. Upload file sertifikat PDF ke folder ini
2. Pastikan nama file sesuai dengan yang digunakan di komponen Experience
3. File akan otomatis terbuka di tab baru ketika link diklik
